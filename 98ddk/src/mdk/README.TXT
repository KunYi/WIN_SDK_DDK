Modem Development Kit Readme
==================================
The Modem Development Kit (MDK) has been moved to the Windows 2000 Driver Development Kit.

The Windows 98 Device Driver Kit contains only the MCT96 Test Suite, for use with all Windows platforms except Windows 2000. See the MCT96 folder for this tool.


The contents of this MDK directory are:

README.TXT     This file; describes files located in the MDK folder.

MCT96          A folder containing the Modem INF testing tool.
