NEC PC-9800 Series Sample Code Notes

This directory contains sample code for drivers of NEC PC-9800 Series.
These drivers work only with NEC PC-9821 Series, don't work with IBM PC-AT Compatible machines.

If you want to use WDEB386.EXE or WDEB98.EXE on NEC PC-9800 series,
please use the one under \src\pc98\bin directory.
