Daytime.vxd:  Sample Kernel Mode TCP Client

[This is preliminary documentation and subject to change.]

Summary

This directory contains project files for Daytime.vxd. This sample demonstrates a simple TDI client written using the DriverNetworks framework. Daytime driver periodically (every 10 seconds) establishes a TCP connection to the DayTime server (U.S. Naval Observatory, Port 13) and receives a GMT timestamp message from the server.

Building the Driver

Open Daytime.dsp in MSVC 6.0 IDE and build it. The build creates files Daytime.vxd and Tcpeco.nms. You will need DriverWorks, DriverNetworks, and the Microsoft Windows DDK installed to successfully build the driver.

Tools

On a Win9x machine, use DriverStudio’s DriverMonitor tool to select and start the Daytime.vxd driver. Watch the debug traces on the DriverMonitor window.
