1.1 Cam1394: DriverWorks 1394 Digital Camera Sample Driver

SUMMARY

Cam1394 sample has 2 drivers for 1394 digital video camera hardware.  The sample was developed using the Orange Micro iBot Firewire Video Conferencing Web Cam.  There are actually 2 drivers.  One is an AVStream filter-centric video capture driver written using the DriverWorks AVStream C++ Framework.  The other driver is a Kernel Streaming video capture driver written using the DriverWorks KStream C++ Framework.  Both drivers make use of DriverWorks 1394 WDM C++ Framework.

The samples are located in the avstream and kstream folders.  Refer to AvCam1394.htm for information about the AVStream filter-centric video capture driver.  Refer to KsCam1394.htm for information about the KStream video capture driver.
